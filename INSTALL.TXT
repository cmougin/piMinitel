How to INSTALL: // testé avec succès le 30/01/2021
immediately after 1st boot strap (and set-up)
1. mkdir /home/pi/Minitel
2. clone la branche MASTER localement à partir de github:
cd /home/pi/Minitel && git clone --branch master https://github.com/cmougin/piMinitel .
3. configure le "service" Minitel en tant que service à initialiser au démarrage (boot)
sudo ln Minitel.service /etc/systemd/system/Minitel.service
sudo chmod 640 /etc/systemd/system/Minitel.service
//pour vérifier: systemctl status Minitel.service => le service est loaded mais disabled
sudo systemctl daemon-reload
sudo systemctl enable Minitel
4. terminer la configuration en liant le shell local et system
sudo ln Minitel.sh /usr/local/bin/Minitel.sh
5. pour tester sans attendre le prochain boot: brancher le Minitel sur USB
// taper sur le clavier du Minitel 2:
Fnct + Sommaire (bascule en mode peri-informatique ie terminal, sortant du menu du Minitel)
Fnct + T (en même temps, puis on relâche), puis A
Fnct + T (en même temps, puis on relâche), puis E
Fnct + P (en même temps, puis on relâche), puis 9 (9600 Bds) 
sudo systemctl start Minitel 
=> l'écran du Minitel affiche un terminal linux et le prompt pi@raspberrypi:~$
6. gérer les accents 
//wget http://canal.chez.com/mntl.ti => ou ne rien faire pour repartir de la version déjà là
tic mntl.ti -o /etc/terminfo
sudo apt-get install screen
sudo cat screenrc >> /etc/screenrc
cat .inputrc >>  ~/.inputrc
7. accrocher le bootstrap.sh au .bashrc pour le lancer à chaque login (https://mars-hack-lab.fr/?p=282)
echo '
# Minitel
    case "$TERM" in
    m1*|m2*|minitel*)
    export LC_ALL=C
    /home/pi/Minitel/bootstrap.sh
    ;;
    esac
    '>> ~/.bashrc
8. éventuellement, installer mopidy
sudo apt install libspotify12 pyspotify
sudo pip3 install modipy-spotify
sudo pip3 install Mopidy-Mofidy
sudo systemctl enable mopidy
* dans /etc/mopidy/mopidy.conf:
    [audio]
    output = alsasink
    [http]
    hostname = 0.0.0.0
    [spotify]
    username = 111…8 (le user spotify)
    password = H…s (le mot de passe spotify)
    client_id = d…a (récupéré ici: https://auth.mopidy.com/spotify/ )
    client_secret = D…0= (de même)
    [mopify]
    enabled = true
    debug = false
sudo systemctl restart mopidy
#vérifier que tout se passe bien:
sudo mopidyctl config|more #pour regarder la config complète en mémoire
sudo journalctl -u mopidy|more #pour regarder les logs qui passent sur ce service
aller sur http://192.168.0.19:6680/mopify (l’IP du minitel, cf ifconfig)
et activer le SERVICE spotify avant d’utiliser la GUI pour lancer la musique
